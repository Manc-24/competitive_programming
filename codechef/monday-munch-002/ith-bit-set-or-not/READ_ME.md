PROBLEM STATEMENT

Is the i-th bit set
You are given two integers, n and i. Your task is to determine whether the i th  bit (0-indexed from the least significant bit) in the binary form of n is 1
Return true if the bit is set, otherwise return false.

Function Declaration
Function Name
isBitSet
isBitSet – This function checks whether the i th  bit (0-indexed from the least significant bit) in the binary representation of an integer n is set.

Parameters
n: An integer whose binary representation is to be examined.
i: An integer indicating the bit position (0-indexed from the least significant bit).
